# 使ってよい Material Symbols 名

アイコンはこのリストから選ぶ。リスト外の名前は**グリフにならず文字列がそのまま画面に出る**
（stills 確認で「英単語がそのまま見えたら」それが原因）。
近いものが無ければ、意味が最も近い汎用名（info, star, category など）で妥協する。
