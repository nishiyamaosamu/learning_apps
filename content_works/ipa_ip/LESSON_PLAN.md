# ITパスポート レッスン計画

全体目標：全25中分類を **100レッスン** を目安に構築する。
単語暗記は別機能（暗記カード）に委譲し、各レッスンは **目標／説明＋頻出概念＋計算・図解** を中心に組み立てる。

- 1レッスン ＝ **約16〜20ページ（＝シーン／1画面）** 前後。
- 構成の基本：導入 → 説明（テキスト/画像/アニメ）を一通り → **末尾にミニクイズをまとめて配置** → 本番演習（exercises）参照。
- 「ページ」は [docs/LESSON.md](../../docs/LESSON.md) の **scene**（1画面）に対応。`説明`＝narrationシーン、`クイズ`＝quizMultipleChoice / quizFillInTheBlank。
- **ビジュアル**列：そのページの表示モード。`アニメ`＝図解アニメ（app側Flutterウィジェット、`animationKey`参照）／`画像`＝静的画像（step の `imageUrl`）／`—`＝テキストのみ。
- **アニメと画像が両立する場合はアニメを優先**する（[docs/LESSON.md](../../docs/LESSON.md) の表示モード規定）。アニメはタップ（step）ごとに phase が進み、説明文と同期して段階展開する。
