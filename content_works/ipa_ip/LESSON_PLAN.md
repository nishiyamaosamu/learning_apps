# ITパスポート レッスン計画

全体目標：全25中分類を **150〜200レッスン** を目安に構築する。
単語暗記は別機能（暗記カード）に委譲し、各レッスンは **目標／説明＋頻出概念＋計算・図解** を中心に組み立てる。

- 1レッスン ＝ 学習ページ **約8〜10**（クイズ除く）＋ クイズ **約2〜3** ＝ **合計約10〜13シーン**。
- 構成の基本：導入 → 説明（テキスト/画像）を一通り → **末尾にミニクイズをまとめて配置**。
- 「ページ」は [docs/LESSON.md](../../docs/LESSON.md) の **scene**（1画面）に対応。`説明`＝narrationシーン、`クイズ`＝quizMultipleChoice / quizFillInTheBlank。
- **ビジュアル**列：そのページの表示モード。`画像`＝静的画像（step の `imageUrl`）／`—`＝テキストのみ。
